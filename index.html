About Me

Hello, I’m Alex, and I love bringing ideas and designs to life on the web!

I am currently diversifying my skillset by studying Git, React, TypeScript and Bootstrap, so that I can build even more interesting and sophisticated websites in the future.

After graduating university, I had the opportunity to work as a musician for a number of premium cruise lines. Throughout this time I got to travel the world, seeing some extraordinary places (the photo is from Bergen in Norway, strongly recommend!) and meeting amazing people.

When I’m not studying or working on my next project, I still love to be around music, whether it’s playing, producing or going to see live gigs.

I’m a big sports fan, especially American Football (I haven’t missed a Packers game for 3 years!), but I also play football regularly.

Projects

Filmby's - Movie Review Site

Filmby's is a website that allows users to view critics' ratings of movies, as well as give their personal ratings and build their own ranked list of their favourite films. As a movie buff, I always found myself checking IMDb, Rotten Tomatoes and Metacritic to see how highly a movie is regarded, but could never find a site where all of those scores were available in the same place. So, I created Filmbys!

I built Filmby's using HTML, CSS/SASS and JavaScript. The JavaScript was written using an MVC style architecture, and contains AJAX (async/await, fetch API), ES6 classes and modules, as well as a lot of data handling and manipulation (including sending and receiving data to/from local storage). The site functions like an app, where the vast majority of the content is dynamically displayed on a single page based on user interaction.

Forever Flora - E-commerce Site

Forever Flora is a demo e-commerce website I built for a terrarium business I started in late 2021. The business currently operates on Etsy, so I wanted to build this to explore the possibility of transitioning away from an established online marketplace, as well as to develop my skills in building a larger, multi-page website.

Forever Flora was built using HTML, CSS/SASS and JavaScript. I put a lot of effort into the responsiveness of the site, which included creating image carousels and a collapsing burger menu that could be triggered by a mixture of CSS media queries/SASS Mixins and JavaScript event listeners.

Profile Site

I build this site as a personal profile page, where user's could view information about me, my coding experience, and the projects I have developed. I chose to build this site from scratch, rather than use a template, to further develop my skills, and so that it could be included in my portfolio of work.

This site was built using HTML, CSS/SASS and JavaScript. The site is all contained on a single page, so to make it engaging I used the JavaScript Intersection Observer API to dynamically style elements based on the users scroll position. This also allowed me to implement lazy-loading images to improve loading times. While building this site I also gained some experience using Figma, as I felt the UX/UI design was particularly important.
